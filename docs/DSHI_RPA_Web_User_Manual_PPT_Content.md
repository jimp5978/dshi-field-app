## 슬라이드 11: 검사신청 시스템

### 📋 검사신청 3단계 워크플로우

**신청 조건**
- ⚙️ **동일 공정**: 모든 선택 항목이 같은 다음 공정이어야 함
- ❌ **미완료 상태**: 1900-01-01 값은 미완료로 처리
- 🚫 **중복 방지**: 이미 신청된 항목은 재신청 불가

**3단계 워크플로우**
```
🟡 대기중 → 🟢 승인됨 → 🔵 확정됨
   ↓         ↓         ↓
Level 1+   Level 2+   Level 3+
신청/취소    승인/거부   확정 처리
```

**자동 처리**
- 🔄 **다음 공정 계산**: 현재 진행도에 따른 정확한 다음 공정 결정
- 🧹 **자동 정리**: 성공한 항목들은 저장리스트에서 자동 제거
- 📊 **상태 추적**: 실시간 신청 상태 업데이트

> 📸 **[스크린샷 자리표시자]**: 검사신청 화면
